# brightness管理框架

（1）查看backlight设备节点
ls /sys/class/backlight/backlight/
（2）查看backlight最大亮度等级
cat /sys/class/backlight/backlight/max_brightness
（3）设置backlight亮度等级
echo 64 > /sys/class/backlight/backlight/brightness
（4）查看backlight当前亮度等级
cat /sys/class/backlight/backlight/actual_brightness
